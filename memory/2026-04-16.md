## Lessons Learned
- When adding Tailwind to Next.js project, must include all three config files and dependencies
- Vercel deployments with new dependencies take 2-3 minutes to fully propagate
- Browser screenshots are essential for diagnosing CSS issues that curl can't reveal
- **Tailwind arbitrary values** (`bg-[#hex]`) are safer than inline styles for dynamic builds
- Always verify sitemap generation after adding new articles (npm run sitemap)
