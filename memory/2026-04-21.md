## Blog Page Auto-Sync (Completed)

**Problem**: Both `app/page.js` (homepage) and `app/blog/page.js` (blog list) used hardcoded article arrays. Every new article required manual updates to 3+ files.

**Solution**: Refactored both pages to auto-import from single source of truth:
- `lib/posts-meta.js` - Single source of truth for all article metadata
- `app/page.js` - `require('../lib/posts-meta')` + `allPosts.slice(0, 6)`
- `app/blog/page.js` - `require('../../lib/posts-meta')` + map over all posts
- Added `allPosts.sort((a, b) => new Date(b.date) - new Date(a.date))` at end of posts-meta.js

**Result**: Future article publishing workflow simplified to:
1. Write article → `content/posts/[slug].md`
2. Add metadata → `lib/posts-meta.js`
3. `npm run build` → `git push`
4. ✅ Homepage and blog auto-sync
